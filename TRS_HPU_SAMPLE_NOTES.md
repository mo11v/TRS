# HPU Sample Added

تم إضافة ملف المعدات التجريبي:

```text
sample_data/equipment/HPU/111470.xlsx
uploads/equipment/HPU/111470.xlsx
sample_data/HPU_import_sample.zip
```

## الهدف
- `111470.xlsx` يمثل معدة واحدة داخل قسم HPU.
- اسم الملف `111470` يتم استخدامه كـ Equipment ID.
- `HPU_import_sample.zip` جاهز للاختبار من شاشة Import Equipment ZIP، وهيكله:

```text
HPU/
  111470.xlsx
```

## ملاحظة
الشيت يحتوي أعمدة صيانة مثل:
- Date & Time Of Maintenance
- Maintenance Details
- Spare Part Number used
- Type Of Maintenance
- Total working HRS
- Work Order No.
- Maintained By
